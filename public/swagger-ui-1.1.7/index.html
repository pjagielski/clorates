DevRates API

This is a read-only REST frontend to what you can find on DevRates main site. Built with Clojure (more details soon...)

Examples

  • List all Java web frameworks 
    GET http://devrates.com/api/projects?tags=java,web framework
  • List all PHP frameworks 
    GET http://devrates.com/api/projects?tags=php
  • List all reviews of Grails 
    GET http://devrates.com/api/projects/Grails/reviews
  • List all releases of Spring 
    GET http://devrates.com/api/projects/Spring Framework/releases
  • Get newest releases 
    GET http://devrates.com/api/releases
